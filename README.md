# 🏥 myClinic Backend

A RESTful API developed for a clinic management system, allowing patients and healthcare professionals to manage appointments, medical exams, analyses and prescriptions.

## About

myClinic is a clinic management platform developed as an academic project for the Bachelor's Degree in Web Information Systems and Technologies.

The system was designed to simplify the management of healthcare services by providing a centralized platform where patients can schedule appointments and healthcare professionals can manage consultations, exams and analyses.

## Features

- Patient registration and authentication
- Doctor management
- Appointment scheduling
- Exam and analysis management
- Medical prescriptions
- Appointment history
- Invoice management
- Doctor and specialty filtering
- Role-based access control
- Secure authentication with JWT

## Technologies

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Jest
- Supertest

## Testing
